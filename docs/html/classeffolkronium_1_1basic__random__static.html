Detailed Description

template<typename Engine, typename Seeder = seeder_default, template< typename > class IntegerDist = std::uniform_int_distribution, template< typename > class RealDist = std::uniform_real_distribution, typename BoolDist = std::bernoulli_distribution>
class effolkronium::basic_random_static< Engine, Seeder, IntegerDist, RealDist, BoolDist >

Base template class for random with static API and static internal member storage.

Note
it is NOT thread safe but more efficient then basic_random_thread_local
Parameters
EngineA random engine with interface like in the std::mt19937
SeederA seeder type which return seed for internal engine through operator()
